1. Estructura y DOM

HTML nos permite estructurar la información en forma de árbol. Esta representación jerárquica se conoce como DOM (Modelo de Objeto del Documento).

Gracias al DOM podemos manipular el contenido, modificar atributos, manejar eventos y crear elementos dinámicamente mediante JavaScript.
